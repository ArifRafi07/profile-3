Tentang Saya

Halo! Nama saya Arif Rafi, sering dipanggil Rafi atau Uping. Saya lahir pada tanggal 7 Oktober 2008. Saya berasal dari Paroto dan saat ini tinggal bersama orang tua saya. Ayah saya adalah Kepala Desa dan Ibu saya adalah perangkat desa. Saya juga memiliki seorang kakak laki-laki.

Saat ini, saya adalah seorang siswa di SMKN 3 Soppeng dengan jurusan TJKT (Teknik Komputer dan Jaringan Telekomunikasi), dan saya aktif dalam program AXIOO. Di luar kegiatan sekolah, saya memiliki minat pada modifikasi sepeda motor, baik motor saya sendiri maupun motor kakak saya. Saya juga sering membantu Ayah dan Kakak dalam instalasi listrik.

Selain itu, hobi saya adalah bermain bola voli, dan saya aktif di kegiatan ekstrakurikuler bola voli serta pramuka. Selama SMP, saya juga pernah menjadi anggota MPK. Saya bersemangat untuk terus belajar dan mengembangkan diri di berbagai bidang.
